# DnD-helper
An app that I built upon to fiddle with things

To start app:
- navigate to ui
- run npm i
- run: npm start

To set up and start server:
- navigate to server
- run npm i
- run: npm start
- For the database: npm run setup


This App uses:
- React.JS
- JavaScript
- Redux
- React Router
- PHP
- HTML/ CSS
- Node.JS
- Express.JS
- Axios
